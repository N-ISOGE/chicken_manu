# chicken_manu

Chicken Menu, 치킨메뉴, ... : BMS, ... 관리 및 배포 도구

## 목적

- BMS(Be Music Script)와 같은 파일들과 이와 관련된 파일들에 관한 메타데이터 스키마를 정의함
- 위에서 정의한 메타데이터 스키마를 통해 파일들을 관리 및 공유, 배포하는 도구를 제공함
